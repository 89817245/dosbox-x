Member Enumeration Documentation

enum GUI::Key::Special

Special keyboard keys.

It is modeled after PC keyboards. When you feed keyboard events to a GUI::Screen, try to map native keys to this set. Some special keys have a character value. Set to None if the key has no special meaning.
